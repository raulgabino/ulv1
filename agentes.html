Instalacion para agentes

Todo lo que necesitas para instalar, verificar y mantener actualizados los skills de Universidad Langosta.

Inicio rapido

1. Instala desde ClawHub

clawhub install universidad-langosta/brt-negociacion

2. Verifica

# Preguntale a tu agente:
Diagnostica a un concesionario Tipo C que pregunta por el fideicomiso.

# Debe responder con el formato del skill:
# Tipo: C -- Tecnicamente curioso + razon
# Focos rojos: ninguno detectado
# Senial clave: ...
# Siguiente movimiento: ...

3. Certifica

Ve a la pagina de certificacion y corre los 4 casos con tu agente.

Instalacion manual

Si prefieres no usar ClawHub o necesitas una instalacion offline:

# 1. Crear estructura de directorios
mkdir -p ~/.openclaw/skills/brt-negociacion/references

# 2. Descargar skill base
curl -o ~/.openclaw/skills/brt-negociacion/SKILL.md \
  https://universidadlangosta.com/raw/brt-negociacion/SKILL.md

# 3. Descargar corpus de modulos
curl -o ~/.openclaw/skills/brt-negociacion/references/modulos.md \
  https://universidadlangosta.com/raw/brt-negociacion/modulos.md

# 4. Verificar que los archivos esten completos
wc -l ~/.openclaw/skills/brt-negociacion/SKILL.md
# Debe mostrar ~192 lineas

wc -l ~/.openclaw/skills/brt-negociacion/references/modulos.md
# Debe mostrar ~222 lineas

Actualizacion y recertificacion

Desde ClawHub

clawhub update universidad-langosta/brt-negociacion

Manual

# Re-descargar ambos archivos (sobreescribe)
curl -o ~/.openclaw/skills/brt-negociacion/SKILL.md \
  https://universidadlangosta.com/raw/brt-negociacion/SKILL.md

curl -o ~/.openclaw/skills/brt-negociacion/references/modulos.md \
  https://universidadlangosta.com/raw/brt-negociacion/modulos.md
Politica de versiones:

Usamos semver. Patch (1.0.x): correcciones menores, no requiere recertificar. Minor (1.x.0): modulos nuevos o ampliados, recertificacion recomendada. Major (x.0.0): cambios estructurales, recertificacion obligatoria.

Estructura del skill

# Estructura de archivos del skill

~/.openclaw/skills/brt-negociacion/
  SKILL.md      # Skill base — frontmatter + instrucciones
  references/
    modulos.md  # Corpus detallado — 7 modulos

SKILL.md

El skill base que OpenClaw carga automaticamente. Define el rol del agente, reglas de comportamiento, herramientas de asistencia y formatos de salida.

Ver SKILL.md

modulos.md

El corpus de referencia con el conocimiento completo: tipologias, fases, seniales, tacticas, definiciones, gestion institucional y conocimiento invisible.

Ver modulos.md
Como lee el agente estos archivos:

OpenClaw carga el SKILL.md automaticamente al detectar que el tema es relevante (por el campo description del frontmatter). El SKILL.md referencia a references/modulos.md que el agente lee bajo demanda cuando necesita el detalle completo de un modulo.
